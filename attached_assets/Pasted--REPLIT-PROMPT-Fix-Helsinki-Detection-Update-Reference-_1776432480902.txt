### 4. Resize uploaded images

After I upload the 3 files, resize them to match existing references:
- Max 1200px on longest side
- JPEG quality 85
- Convert PNG to JPG if needed

```python
from PIL import Image
from io import BytesIO

for fname in ['helsinki_lineup.jpg', 'shelf_ref_21.jpg', 'shelf_ref_22.jpg']:
    path = f'backend/photo_report/references/{fname}'
    img = Image.open(path)
    if max(img.size) > 1200:
        ratio = 1200 / max(img.size)
        new_size = (int(img.size[0] * ratio), int(img.size[1] * ratio))
        img = img.resize(new_size, Image.LANCZOS)
    buf = BytesIO()
    img.convert("RGB").save(buf, format="JPEG", quality=85)
    with open(path, 'wb') as f:
        f.write(buf.getvalue())
    print(f"Optimized {fname}: {img.size}")
```
