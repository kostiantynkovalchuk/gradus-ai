   git add backend/static/pulse_videos/*.mp4
   git add backend/static/pulse_memes/*.png

   du -h backend/static/pulse_videos/*.mp4
   du -h backend/static/pulse_memes/*.png
   Files over 100MB need git-lfs. Under 100MB push normally.
