## Context

Alex Photo Report is a Telegram bot for AVTD (Торговий Дім АВ, Ukraine) that verifies merchandising compliance at retail stores using Claude Vision API.

**Stack:** FastAPI/Render, Neon PostgreSQL, python-telegram-bot, Claude Vision API
**Repo:** github.com/kostiantynkovalchuk/gradus-ai
**Bot files:** `backend/photo_report/`
**Current state:** Migration 023 applied. Issues 1-11 from previous sprint all fixed. Deterministic scoring engine (`scoring.py`) is live. Photos stored as BYTEA in Neon. `vision_raw_json` saved per report. scoring_version=v2.

We ran Round 2 of expert testing (11 photos, expert Alexandr provided ground-truth corrections). Results: **44% accuracy** (12/27 measurable checks correct). Shelf talker false positives are fixed (100% correct). But brand detection at distance and category recognition in dense stores are still failing.

This sprint fixes the remaining accuracy issues and adds an expert feedback loop for continuous improvement.

## PHASE 1: SYSTEM_PROMPT Improvements

### 1.1 — Confidence Gating

The biggest single issue: Claude reports 0% for categories it simply can't see clearly, rather than admitting uncertainty. This causes false auto-fails and misleading reports.

**Add to the SYSTEM_PROMPT output JSON schema:**

```
For EACH category (vodka, cognac, wine, sparkling, soju), report:

  "confidence": "high" | "medium" | "low"

Rules:
  "high" = I can clearly read brand labels, confident in my count
  "medium" = I can identify brands by bottle shape/color but labels are blurry or distant
  "low" = bottles are too small, distant, angled, or obscured to count reliably

  CRITICAL: If confidence is "low", set facing counts to null (not 0).
  
  0 means: "I carefully looked and this category is ABSENT from the shelf."
  null means: "I cannot reliably assess this category from this photo."
  
  Only report 0 if you are CERTAIN the category has zero AVTD products.
  If you're not sure → null.
```

**Update `scoring.py` to handle null:**

```python
def calculate_share(our_facings, total_facings, confidence):
    if confidence == "low" or our_facings is None:
        return {"share": None, "assessed": False}
    if total_facings == 0:
        return {"share": 0, "assessed": True}
    return {"share": round(our_facings / total_facings * 100), "assessed": True}
```

**Update report formatting:**
- If assessed: show percentage as normal → "✅ Горілка: 56% (норма ≥25%)"
- If not assessed: show info note → "📊 Вино: не вдалося оцінити (фото занадто далеке)"

### 1.2 — Row-by-Row Shelf Scanning Method

Instead of scanning the whole photo at once (which causes Claude to focus on the most visually prominent items and miss the rest), instruct it to scan systematically.

**Add to SYSTEM_PROMPT:**

```
ANALYSIS METHOD — mandatory row-by-row scanning:

Step 1: Count the number of horizontal shelf rows visible in the photo.
Step 2: For EACH row, starting from TOP to BOTTOM:
  - Describe what category is on this row (vodka? cognac? wine? imported spirits?)
  - List every brand you can identify, left to right
  - Count facings per brand

Step 3: After scanning ALL rows, compile totals per category.

IMPORTANT: Do NOT skip any row. Even if a row looks like it only has competitors, 
scan it anyway — AVTD brands may be mixed in.

Example output for a 4-shelf display:
  "shelf_scan": [
    {"row": 1, "category": "imported spirits", "brands": ["Jameson x2", "VAT69 x1", "Bell's x1"]},
    {"row": 2, "category": "cognac", "brands": ["ADJARI 3★ x3", "ADJARI 5★ x2", "ADJARI Cherry x1", "Aznauri x2"]},
    {"row": 3, "category": "vodka", "brands": ["GreenDay Classic x4", "Ukrainka x3", "Nemiroff x2", "Khortytsia x3"]},
    {"row": 4, "category": "wine + soju", "brands": ["Villa UA Rosé x2", "Villa UA White x1", "Funju x2", "competitor wine x5"]}
  ]
```

This forces Claude to look at every shelf instead of glancing at the photo and guessing.

### 1.4 — Partial Photo Awareness

When a photo clearly shows only a portion of the shelf (cut off at bottom, sides cropped), Claude should note this rather than treating the visible area as the complete store.

**Add to SYSTEM_PROMPT:**

```
PHOTO COMPLETENESS CHECK:
Before counting, assess whether the photo shows:
  - Full shelf zone (both edges visible) → "complete"
  - Partial view (shelf clearly continues beyond frame) → "partial"
  - Close-up of one section → "close_up"

Report in JSON:
  "photo_completeness": "complete" | "partial" | "close_up"

If "partial" or "close_up":
  - Add note: "Фото показує лише частину полиць — результати можуть бути неповними"
  - Set confidence to "medium" maximum for share calculations
  - Do NOT report 100% share just because you only see our brands in a close-up
```

### Loading reference images at startup

```python
# references.py — load once at startup, reuse for every analysis
import base64
from pathlib import Path
from PIL import Image
from io import BytesIO

REFERENCES_DIR = Path(__file__).parent / "references"
MAX_SIZE = 1200  # resize to max 1200px on longest side
QUALITY = 85     # JPEG compression quality

def _load_and_optimize(path: Path) -> str:
    """Load image, resize if needed, return base64 string."""
    img = Image.open(path)
    
    # Resize if too large (saves tokens)
    if max(img.size) > MAX_SIZE:
        ratio = MAX_SIZE / max(img.size)
        new_size = (int(img.size[0] * ratio), int(img.size[1] * ratio))
        img = img.resize(new_size, Image.LANCZOS)
    
    # Convert to JPEG, compress
    buf = BytesIO()
    img.convert("RGB").save(buf, format="JPEG", quality=QUALITY)
    return base64.b64encode(buf.getvalue()).decode()

def load_references() -> list[dict]:
    """Load all reference images from disk. Called once at startup."""
    refs = []
    for img_path in sorted(REFERENCES_DIR.glob("*.jpg")):
        b64 = _load_and_optimize(img_path)
        brand = img_path.stem  # e.g. "greenday_classic_500"
        refs.append({
            "brand": brand,
            "b64": b64,
            "media_type": "image/jpeg"
        })
    return refs

# Module-level: loaded once when the module is imported
BRAND_REFERENCES = load_references()
```

```python
# vision.py — updated analyze_photos function
from references import BRAND_REFERENCES

async def analyze_photos(photos_b64: list[str]) -> dict:
    """Analyze shelf photos with reference images as context."""
    
    content = []
    
    # Step 1: Reference images FIRST
    content.append({
        "type": "text",
        "text": "REFERENCE IMAGES — these are AVTD brand products. Memorize how each brand looks:"
    })
    
    for ref in BRAND_REFERENCES:
        content.append({
            "type": "image",
            "source": {"type": "base64", "media_type": ref["media_type"], "data": ref["b64"]}
        })
        # Label each reference
        label = ref["brand"].replace("_", " ").title()
        content.append({
            "type": "text",
            "text": f"↑ This is {label}. Look for this brand on the shelf photos below."
        })
    
    content.append({
        "type": "text",
        "text": "---\nNow analyze the following shelf photo(s). Find the brands shown above."
    })
    
    # Step 2: Actual shelf photos
    for b64 in photos_b64:
        content.append({
            "type": "image",
            "source": {"type": "base64", "media_type": "image/jpeg", "data": b64}
        })
    
    # Step 3: Analysis prompt (SYSTEM_PROMPT with all instructions)
    content.append({
        "type": "text",
        "text": ANALYSIS_PROMPT  # the full system prompt with row-by-row scanning, confidence, etc.
    })
    
    messages = [{"role": "user", "content": content}]
    
    response = await call_claude_vision(messages)
    return parse_json(response)
```

### Automated benchmark test script

```python
# test_benchmark.py — run after each prompt/scoring change
import json
from pathlib import Path

BENCHMARK_DIR = Path(__file__).parent / "test_benchmark"
TOLERANCE = 10  # ±10% is acceptable

async def run_benchmark():
    """Run all 11 test photos through the analysis pipeline and compare with expert corrections."""
    
    expected = json.loads((BENCHMARK_DIR / "expected_results.json").read_text())
    results = []
    
    for i in range(1, 12):
        photo_key = f"photo_{i:02d}"
        photo_path = BENCHMARK_DIR / f"{photo_key}.jpg"
        
        if not photo_path.exists():
            continue
        
        # Run analysis
        with open(photo_path, "rb") as f:
            photo_bytes = f.read()
        b64 = base64.b64encode(photo_bytes).decode()
        
        vision_result = await analyze_photos([b64])
        score_result = calculate_score(vision_result)
        
        # Compare with expected
        exp = expected.get(photo_key, {})
        comparison = {"photo": photo_key, "checks": []}
        
        for cat in ["vodka", "wine", "cognac", "sparkling"]:
            exp_val = exp.get(f"{cat}_share")
            ai_val = score_result["shelf_shares"].get(cat)
            
            if exp_val is None:
                continue  # expert didn't assess this category
            
            if ai_val is None:
                # AI said "low confidence" — better than wrong 0%
                comparison["checks"].append({
                    "category": cat,
                    "expected": exp_val,
                    "actual": "not_assessed",
                    "pass": True,  # not assessed is acceptable
                    "note": "AI correctly flagged low confidence"
                })
            else:
                deviation = abs(ai_val - exp_val)
                comparison["checks"].append({
                    "category": cat,
                    "expected": exp_val,
                    "actual": ai_val,
                    "deviation": deviation,
                    "pass": deviation <= TOLERANCE
                })
        
        results.append(comparison)
    
    # Summary
    total_checks = sum(len(r["checks"]) for r in results)
    passed = sum(1 for r in results for c in r["checks"] if c["pass"])
    
    print(f"\n{'='*50}")
    print(f"BENCHMARK RESULTS: {passed}/{total_checks} passed ({round(passed/total_checks*100)}%)")
    print(f"{'='*50}")
    
    for r in results:
        for c in r["checks"]:
            status = "✅" if c["pass"] else "❌"
            print(f"  {status} {r['photo']} {c['category']}: expected={c['expected']}, actual={c['actual']}")
    
    return {"total": total_checks, "passed": passed, "details": results}
```

## PHASE 2: Code-Level Accuracy Improvements

### 2.1 — Two-Pass Retry for Low-Confidence Categories

When Claude reports `"confidence": "low"` for any category, automatically trigger a focused second API call.

**New function in `vision.py`:**

```python
async def analyze_focused(photos_b64: list[str], focus_category: str) -> dict:
    """Re-analyze photos focusing on a single category."""
    
    focus_prompts = {
        "wine": "IGNORE vodka and cognac. Look ONLY for wine bottles. Scan every shelf row for standard wine-shaped bottles (750ml, taller/thinner than vodka). Look for Villa UA (medallion label), Didi Lari, Kristi Valley, Kosher. Count ALL wine bottles including competitors.",
        "cognac": "IGNORE vodka and wine. Look ONLY for cognac/brandy bottles. Scan every shelf row for ADJARI (dark bottles, golden caps, diamond mesh), DOVBUSH, KLINKOV, ЖАН-ЖАК. Count ALL cognac bottles including competitors.",
        "sparkling": "IGNORE still wine and vodka. Look ONLY for sparkling wine bottles (champagne-shaped, dark bottles with foil neck wrapping). Look for Villa UA Sparkling. Count ALL sparkling bottles including competitors.",
        "vodka": "IGNORE cognac and wine. Look ONLY for vodka bottles. Look especially for UKRAINKA (white label, clear bottle) and HELSINKI (blue bottle/label) — these are commonly missed. Also count ALL competitor vodka. Scan every shelf row."
    }
    
    messages = [
        {"role": "user", "content": [
            *[{"type": "image", "source": {"type": "base64", "media_type": "image/jpeg", "data": b64}} for b64 in photos_b64],
            {"type": "text", "text": f"FOCUSED RE-ANALYSIS: {focus_prompts[focus_category]}\n\nReturn JSON with ONLY the {focus_category} section."}
        ]}
    ]
    
    # Call Claude with focused prompt
    result = await call_claude_vision(messages)
    return parse_json(result)
```

**Integrate into main analysis flow:**

```python
async def analyze_and_score(photos_b64, photos_bytes, store_name, user_id):
    # Pass 1: Full analysis
    vision_result = await analyze_photos(photos_b64)
    
    # Pass 2: Retry low-confidence categories
    retried_categories = []
    for category in ["vodka", "wine", "cognac", "sparkling"]:
        cat_data = vision_result.get(category, {})
        if cat_data.get("confidence") == "low":
            retry = await analyze_focused(photos_b64, category)
            if retry and retry.get("confidence") in ("high", "medium"):
                vision_result[category] = retry
                retried_categories.append(category)
    
    if retried_categories:
        logger.info(f"Retried categories: {retried_categories}")
    
    # Score using deterministic engine
    score_result = calculate_score(vision_result)
    
    # Store everything
    report_id = await save_report(store_name, user_id, vision_result, score_result)
    await save_photos(report_id, photos_bytes)
    
    return report_id, score_result
```

**Cost control:** Only retry when confidence is "low". Typical case: 0-1 retries per report. Worst case: 4 retries. At ~$0.03 per retry call (Haiku), max extra cost is $0.12 per report.

```python
    """Flag suspicious counts that may indicate detection errors."""
    warnings = []
```

**Correction parser — flexible format:**

```python
def parse_correction(text: str) -> dict:
    """Parse expert correction text. Supports flexible Ukrainian/Russian formats.
    
    Examples:
        "Водка - 56%, коньяк - 100%"
        "водки нет, вино 40%, коньяк 23%"
        "Водка правильно, вино - 6%, коньяк - 100%"
        "все правильно"
    """
    result = {}
    text_lower = text.lower()
    
    # Map category keywords to JSON keys
    category_patterns = {
        "vodka": [r"водк[аи]", r"горілк[аи]", r"vodka"],
        "wine": [r"вин[оа]", r"wine"],
        "cognac": [r"коньяк\w*", r"cognac", r"бренді"],
        "sparkling": [r"ігрист\w*", r"игрист\w*", r"sparkling"],
    }
    
    for key, patterns in category_patterns.items():
        for pattern in patterns:
            # Look for "category - XX%" or "category XX%"
            match = re.search(
                pattern + r'\s*[-—:=]?\s*(\d{1,3})\s*%',
                text_lower
            )
            if match:
                result[key] = int(match.group(1))
                break
            
            # Look for "category нет" / "category 0" / "категории нет"
            match_zero = re.search(
                pattern + r'\s*[-—:=]?\s*(нет|немає|відсутн|0\b)',
                text_lower
            )
            if match_zero:
                result[key] = 0
                break
            
            # Look for "category правильно" (means AI was correct)
            match_correct = re.search(
                pattern + r'\s*[-—]?\s*(правильно|вірно|correct|ок)',
                text_lower
            )
            if match_correct:
                result[key] = "correct"  # will be resolved to AI value
                break
    
    # Handle "все правильно" / "all correct"
    if re.search(r'(все|всё|all)\s*(правильно|вірно|correct|ок)', text_lower):
        for key in category_patterns:
            if key not in result:
                result[key] = "correct"
    
    return result
```

```python
@app.get("/hr/api/accuracy")
async def get_accuracy_metrics(days: int = 30):
    """Calculate AI accuracy based on expert corrections."""
    
    query = """
    SELECT 
        pr.id as report_id,
        pr.vision_raw_json,
        pr.score_data,
        ec.corrected_vodka_share,
        ec.corrected_wine_share,
        ec.corrected_cognac_share,
        ec.corrected_sparkling_share,
        ec.created_at
    FROM photo_reports pr
    JOIN expert_corrections ec ON ec.report_id = pr.id
    WHERE ec.created_at >= NOW() - INTERVAL '%s days'
    ORDER BY ec.created_at DESC
    """
    
    rows = await db.fetch(query, days)
    
    metrics = {"vodka": [], "wine": [], "cognac": [], "sparkling": []}
    
    for row in rows:
        score_data = row["score_data"] or {}
        shares = score_data.get("shelf_shares", {})
        
        for cat in metrics:
            ai_value = shares.get(cat)
            expert_value = row[f"corrected_{cat}_share"]
            
            if expert_value is not None and expert_value != "correct" and ai_value is not None:
                deviation = abs(ai_value - expert_value)
                metrics[cat].append({
                    "report_id": row["report_id"],
                    "ai": ai_value,
                    "expert": expert_value,
                    "deviation": deviation
                })
    
    summary = {}
    for cat, deviations in metrics.items():
        if deviations:
            avg_dev = sum(d["deviation"] for d in deviations) / len(deviations)
            exact_matches = sum(1 for d in deviations if d["deviation"] <= 5)
            summary[cat] = {
                "samples": len(deviations),
                "avg_deviation": round(avg_dev, 1),
                "accuracy_within_5pct": round(exact_matches / len(deviations) * 100),
                "worst_miss": max(d["deviation"] for d in deviations),
                "details": deviations[-10:]  # last 10
            }
        else:
            summary[cat] = {"samples": 0, "avg_deviation": None}
    
    return {
        "period_days": days,
        "total_corrections": len(rows),
        "categories": summary
    }
```
