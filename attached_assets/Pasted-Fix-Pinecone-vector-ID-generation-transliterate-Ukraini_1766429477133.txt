Fix Pinecone vector ID generation - transliterate Ukrainian/Cyrillic characters to ASCII.

CRITICAL BUG:
Pinecone rejects vector IDs with non-ASCII characters. Product section names like "Продукція", "Асортимент", "НАШІ ВИНА" cause ingestion failures. Need to sanitize/transliterate section names to ASCII.

FIX in backend/scripts/batch_ingest_websites.py:

Add transliteration function after imports (around line 18):
```python
import re
from unidecode import unidecode

def sanitize_vector_id(text: str) -> str:
    """
    Convert text to ASCII-safe string for Pinecone vector IDs.
    Handles Ukrainian/Cyrillic characters.
    """
    # Use unidecode to transliterate Unicode to ASCII
    ascii_text = unidecode(text)
    
    # Replace spaces with underscores
    ascii_text = ascii_text.replace(' ', '_')
    
    # Remove any remaining non-ASCII/non-alphanumeric except underscore and hyphen
    ascii_text = re.sub(r'[^a-zA-Z0-9_-]', '', ascii_text)
    
    # Limit length to avoid issues
    if len(ascii_text) > 50:
        ascii_text = ascii_text[:50]
    
    return ascii_text
```
