### Fix 2 — /alex dashboard: add alex_avtd recent-queries endpoint
In the file serving /alex routes, add:

```python
@alex_router.get("/alex/api/recent-queries")
async def alex_avtd_recent_queries(limit: int = 50, db: Session = Depends(get_db)):
    from sqlalchemy import text
    rows = db.execute(text("""
        SELECT 
            user_name,
            query,
            preset_matched,
            response_time_ms,
            created_at
        FROM hr_query_log
        WHERE bot_source = 'alex_avtd'
        ORDER BY created_at DESC
        LIMIT :limit
    """), {"limit": limit}).fetchall()
    return [
        {
            "user_name": r[0] or "Anonymous",
            "query": r[1],
            "preset_matched": r[2],
            "response_time_ms": r[3],
            "created_at": r[4].isoformat() if r[4] else None
        }
        for r in rows
    ]
```
