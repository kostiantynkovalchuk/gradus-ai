=== CHANGE 3: ROI FORMULA ===

## Backend: Replace ROI calculation in hunt-analytics endpoint

OLD formula:
  cost_saved_usd = total_hires * 385
  hours_saved = total_vacancies * 2.67
  cost_per_hire_maya = total_vacancies * 0.15 / max(total_hires, 1)

NEW formula (realistic, HR-verifiable):

# Configurable constants (add to hunt_config.py)
"roi": {
    "hr_hourly_rate_uah": 150,        # HR specialist hourly rate
    "hours_per_vacancy_traditional": 3, # Hours HR spends manually per vacancy
    "portal_cost_per_vacancy_uah": 800, # Average cost to post on job portals
    "maya_api_cost_per_search_uah": 6,  # Claude Haiku API cost per full search (~$0.15)
    "maya_time_per_vacancy_minutes": 5,  # Time HR spends with Maya per vacancy
}

# Calculation
traditional_cost_per_vacancy = (hours_trad * hr_rate) + portal_cost  # 450 + 800 = 1250 UAH
maya_cost_per_vacancy = maya_api_cost  # 6 UAH
savings_per_vacancy = traditional_cost_per_vacancy - maya_cost_per_vacancy  # 1244 UAH

total_savings_uah = total_vacancies * savings_per_vacancy
total_savings_usd = total_savings_uah / usd_rate
hours_saved = total_vacancies * (hours_trad - maya_minutes/60)  # vacancies * 2.92 hours
cost_per_hire_maya_uah = (total_vacancies * maya_api_cost) / max(total_hires, 1)
cost_per_hire_traditional_uah = traditional_cost_per_vacancy
